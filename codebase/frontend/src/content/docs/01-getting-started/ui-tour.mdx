목록에서는 다음 동작을 할 수 있어요.

- **검색**: 이름으로 필터링해요.
- **활성/비활성 토글**: 행 왼쪽 동그라미 표시로 즉시 끄고 켜요. 비활성 상태에서는 트리거가 들어와도 실행되지 않아요.
- **더보기(⋮)**: 편집, 복제, 내보내기(JSON), 삭제 등을 선택해요.
- **항목 클릭**: 해당 워크플로우의 **캔버스 에디터**로 이동해요.

**에디터의 Empty State**: 트리거 외의 노드가 아직 없는 워크플로우에서는 캔버스 **우측 상단**에 "시작하기" 카드가 떠요. 여기에서 매뉴얼의 해당 섹션으로 바로 이동할 수 있어요. 첫 비트리거 노드를 추가하면 카드가 자연스럽게 사라져요.
</Callout>

## 팁 & 참고

- 에디터 작업 중 매뉴얼 링크를 클릭하면 **새 탭**으로 열려요. 작업 맥락을 잃지 않도록 설계돼 있어요.
- 파워유저 팁: 노드 설정 패널은 개별 폼과 **JSON/코드 편집 모드**를 상호 전환할 수 있어요. 여러 필드를 한 번에 붙여넣거나 두 노드 설정을 비교할 때 편해요.
- 다음 단계는 [첫 워크플로우 만들기](/docs/01-getting-started/first-workflow)예요. 실제 노드를 올리고 결과를 확인해 봐요.
- 노드 종류가 궁금하면 [노드 개요](/docs/02-nodes/overview)를 참고해요.
