이 페이지에서는 워크플로우 에디터에서 쓸 수 있는 키보드 단축키를 목적별로 묶어 한 표로 정리해요. 자주 쓰는 동작일수록 마우스보다 키보드가 훨씬 빨라요.

<Callout type="note">
`Ctrl+C`/`Ctrl+V`/`Ctrl+D`/`Ctrl+A`는 이름·표현식 같은 텍스트 입력 필드에 포커스가 있을 때는 동작하지 않아요. 텍스트를 자유롭게 입력할 수 있도록 하기 위해서예요. 저장·실행 취소·다시 실행·AI 어시스턴트 토글은 입력 필드에서도 그대로 동작해요.
</Callout>

## 팁 & 참고

- 뷰포트 이동·줌·팔레트로 노드 추가하는 방법은 [캔버스 다루기](/docs/03-workflow-editor/canvas-basics)에서 자세히 다뤄요.
- 노드 선택·복사·삭제·비활성화의 세부 동작은 [노드 편집과 정리](/docs/03-workflow-editor/editing-nodes)에서 확인해요.
- `Cmd/Ctrl + /`로 열고 닫는 AI 어시스턴트의 전체 흐름은 [AI 어시스턴트 개요](/docs/03-workflow-editor/ai-assistant)에서 자세히 다뤄요.
- 에디터 전반의 화면 구성은 [워크플로우 에디터 개요](/docs/03-workflow-editor/overview)에서 큰 그림으로 볼 수 있어요.
