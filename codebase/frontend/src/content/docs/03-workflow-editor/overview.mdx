이 페이지에서는 워크플로우 에디터가 어떤 공간이고 화면이 어떻게 나뉘어 있는지 살펴봐요. 워크플로우 에디터는 캔버스 위에 노드를 놓고 연결선으로 이어서 자동화 로직을 시각적으로 구성하는 화면이에요. 각 영역의 자세한 조작법은 이어지는 페이지에서 다뤄요.

## 에디터의 다섯 영역

- **헤더** (상단) — 뒤로가기(←)로 [워크플로우 목록](/workflows)에 돌아가고, 가운데의 워크플로우 이름을 클릭하면 바로 고칠 수 있어요. 저장 상태 텍스트, Undo/Redo, `🤖` AI 어시스턴트 토글, `Save` 버튼, `▶ Run` 버튼(드롭다운으로 실행 옵션 선택), `⋮` 더보기(버전 히스토리·실행 히스토리·내보내기·삭제)가 한 줄에 모여 있어요.
- **노드 팔레트** (좌측) — 카테고리별 노드 목록, 검색, 최근 사용한 노드를 보여줘요. 자세한 사용법은 [캔버스 다루기](/docs/03-workflow-editor/canvas-basics)에서 이어져요.
- **캔버스** (중앙) — 노드를 배치하고 연결선으로 잇는 작업 공간이에요. 노드 조작은 [노드 편집과 정리](/docs/03-workflow-editor/editing-nodes)에서, 연결선은 [노드 연결하기](/docs/03-workflow-editor/connecting-nodes)에서 자세히 다뤄요.
- **설정 패널 / AI 어시스턴트 패널** (우측, 조건부) — 노드를 클릭하면 설정 패널이 열리고, `🤖` 버튼을 누르면 AI 어시스턴트 패널이 열려요. 둘은 같은 자리를 쓰는 사이라 동시에 열리지 않아요. 자세한 내용은 [노드 설정 패널](/docs/03-workflow-editor/settings-panel)과 [AI 어시스턴트 개요](/docs/03-workflow-editor/ai-assistant)에서 이어져요.
- **Run Results 드로어** (하단, 실행 시) — 워크플로우를 실행하면 화면 아래에 열리고, 노드별 실행 결과를 확인할 수 있어요. 실행 방법은 [실행하기](/docs/05-run-and-debug/running-a-workflow)에서 다뤄요.

<Callout type="note">설정 패널과 AI 어시스턴트 패널은 같은 우측 슬롯을 나눠 써요. 노드를 선택하면 어시스턴트가 닫히고, 어시스턴트를 열면 노드 선택이 풀려요.</Callout>

## 에디터에 들어가고 나가기

- [워크플로우 목록](/workflows)에서 항목을 클릭하면 그 워크플로우의 에디터로 들어가요.
- 목록에서 **워크플로우 만들기**를 누르면 빈 캔버스로 바로 들어가요. 이때 캔버스에는 수동 트리거 노드 1개가 이미 놓여 있어서 바로 다음 노드를 이어 붙이면 돼요.
- 뒤로가기(←)를 누르면 다시 [워크플로우 목록](/workflows)으로 돌아가요.

저장하지 않은 변경 사항이 있으면 헤더의 상태 텍스트가 "저장되지 않은 변경 사항"으로 알려 줘요(저장 중에는 "저장 중...", 저장이 끝나면 "저장됨"으로 바뀌어요). 목록으로 나가기 전에 이 표시를 확인하고 저장해 두면 안전해요.

## 저장 방식 한눈에

- **수동 저장**: 헤더의 `Save` 버튼 또는 `Ctrl+S`(Mac은 `Cmd+S`)로 즉시 저장해요. 저장할 변경 사항이 없으면 버튼이 비활성 상태예요.
- **실행 직전 자동 저장**: `▶ Run`으로 워크플로우를 실행하면, 저장하지 않은 변경 사항이 있을 때 실행 직전에 먼저 저장돼요. 그래서 실행은 항상 캔버스의 최신 상태로 진행돼요.
- 저장과 버전 히스토리의 관계, 워크플로우 JSON 가져오기/내보내기는 [저장·자동저장·가져오기/내보내기](/docs/03-workflow-editor/saving-and-sharing)에서 자세히 다뤄요.

## 빈 캔버스에서 시작할 때

새 워크플로우는 트리거 노드 1개만 있는 상태로 시작해요. 트리거 다음에 이어 붙일 노드가 아직 없으면 캔버스 우측 상단에 "워크플로우를 이어서 완성해봐요" 안내 카드가 나타나요.

- 카드에는 3단계 체크리스트(① 팔레트에서 다음 노드를 드래그 ② 트리거 출력 포트에 연결 ③ 실행해서 결과 확인)와, 각 단계 옆 "자세히" 링크, 그리고 [첫 워크플로우 만들기](/docs/01-getting-started/first-workflow)로 이동하는 "시작 가이드 열기" 버튼이 있어요.
- 트리거가 아닌 첫 노드를 추가하면 카드가 서서히 사라져요. 카드 우측 상단의 닫기 버튼으로 직접 닫을 수도 있어요.
