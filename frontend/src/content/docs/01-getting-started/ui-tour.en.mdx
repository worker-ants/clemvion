Things you can do from the list:

- **Search**: filter by name.
- **Active/inactive toggle**: the circle on the left flips instantly. When a workflow is inactive, even incoming triggers won't run it.
- **More (⋮)**: edit, duplicate, export (JSON), delete.
- **Click an item**: jump to that workflow's **canvas editor**.

  <Step>**Node palette (left)**: nodes grouped by category — Trigger · Logic · Flow · AI · Integration · Data · Presentation. The search at the top jumps to a node by name.</Step>
  <Step>**Canvas (center)**: drag nodes from the palette to place them, then draw a **connection** from an output port to the next node's input port. Scroll to zoom, drag empty space to pan.</Step>
  <Step>**Settings panel (right)**: opens when you click a node. Fields accept `{{ }}` expressions with autocomplete. Changes apply immediately.</Step>

## Tips & notes

- Clicking a manual link while you're in the editor opens it in a **new tab** — by design, so you don't lose your work context.
- Power-user tip: the settings panel can flip between per-field forms and a **JSON/code edit mode**. Handy for pasting many fields at once, or diffing two node configs.
