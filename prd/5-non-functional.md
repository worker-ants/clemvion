## 1. 성능

| ID | 요구사항 | 우선순위 | Phase |
|----|----------|----------|-------|
| NF-PF-01 | 워크플로우 에디터 캔버스는 100개 이상의 노드가 있어도 부드러운 렌더링(60fps) 유지 | 필수 | 1 |
| NF-PF-02 | 워크플로우 저장/로드 응답 시간 2초 이내 | 필수 | 1 |
| NF-PF-03 | 노드 실행 시작 지연(Latency) 최소화 — 실행 엔진 노드 간 핸드오프 100ms 이내 | 필수 | 1 |
| NF-PF-04 | 대량 데이터(10,000건 이상) 처리 시에도 메모리 사용량 안정적 유지 | 필수 | 1 |
| NF-PF-05 | 동시 실행 중인 워크플로우 간 성능 격리 | 필수 | 1 |
| NF-PF-06 | API 응답 시간 p95 < 500ms (CRUD 기준) | 필수 | 1 |
