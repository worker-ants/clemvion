## 1. 듀얼 라이선스 안내 (Dual Licensing)

clemvion("소프트웨어")은 다음 두 가지 라이선스로 제공됩니다.

**AGPL v3 (무료)**
소스코드 공개 의무를 준수하는 조건으로 무료 사용이 가능합니다.
내부 사용 목적은 공개 의무가 없습니다.

**상업 라이선스**
AGPL v3 의무를 면제받고 소스 비공개 상태로 상업적 서비스를
운영하려는 경우 필요합니다.

---

## 2. 상업 라이선스가 필요한 경우 (When You Need One)

AGPL v3의 의무가 부담되는 다음과 같은 경우,
상업 라이선스가 필요합니다. 아래는 예시이며 이에 한정되지
않습니다.

- 소스코드를 공개하지 않고 서비스를 운영하려는 경우
- 최종 사용자에게 소프트웨어의 기능을 직·간접적으로
  제공(SaaS, 백엔드 연동, 설정 UI 연동 등)하려는 경우
- 화이트라벨 또는 OEM 형태로 제품에 내재화하려는 경우

> 자사 임직원이 내부 업무 목적으로만 사용하는 경우에는
> AGPL v3 하에서 무료로 사용 가능하며, 상업 라이선스가
> 필요하지 않습니다.

부여 범위, 허용 사용 형태, 요금, 계약 기간 등 구체적인
조건은 사용 형태와 규모에 따라 결정됩니다.

**문의: admin@getit.co.kr**

## 3. AGPL 의무 면제 (AGPL Exemption)

상업 라이선스 체결 시 라이선시는 다음 의무를 면제받습니다.

a) 네트워크를 통한 서비스 제공 시 소스코드 공개 의무 (AGPL §13)

b) 파생 저작물의 AGPL 조건 하 배포 의무

> 단, 라이선시가 소프트웨어를 AGPL 조건으로 별도 배포하는 것은
> 상업 라이선스와 무관하게 가능합니다.
