### 발견사항

해당 없음

이번 변경사항은 전적으로 다음 범위에 속합니다:

- React Flow가 측정한 노드 `width`/`height` 값을 DTO → `ShadowWorkflow`(인메모리) → LLM 시스템 프롬프트까지 전달하는 데이터 파이프라인 추가
- `ShadowWorkflow`는 코드 주석에 명시된 것처럼 *"never touches the database"* 인메모리 복제본
- `persistAssistantTurn` 경로는 변경되지 않았으며, `width`/`height`는 DB에 저장되지 않음
- 스키마 변경, 마이그레이션, ORM 엔티티, 쿼리 로직, 커넥션 관리 등 데이터베이스 관련 코드는 일체 포함되지 않음

### 요약

변경된 모든 파일은 UI 측정값을 LLM 프롬프트 레이아웃 계산에 활용하기 위한 인메모리 데이터 흐름과 타입 정의 변경에 국한된다. 데이터베이스 접근 계층에는 전혀 영향을 주지 않으므로 데이터베이스 관점의 리뷰 대상에 해당하지 않는다.
