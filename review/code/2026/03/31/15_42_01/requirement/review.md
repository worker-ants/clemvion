### 요약

두 파일은 프론트엔드 테스트 환경 설정을 담당합니다. `vitest.config.ts`는 jsdom 환경, 전역 변수, setup 파일 경로, 테스트 파일 패턴, 경로 별칭을 올바르게 구성하고 있으며, `setup.ts`는 `@testing-library/jest-dom`의 vitest 전용 매처를 임포트하고 있습니다. 요구사항 관점에서 테스트 인프라 설정으로서의 역할을 충실히 수행하고 있으며, 비즈니스 로직이 포함되지 않은 설정 파일이므로 기능 완전성 측면의 별도 검토 사항은 없습니다.
